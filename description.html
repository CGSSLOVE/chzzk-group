치지직 스트리머 그룹 도우미

팔로잉하는 치지직 스트리머를 그룹별로 정리하고 관리하세요.

메인 사이드바에서 가능한 기능

  • drag_indicator

    그룹 및 스트리머 순서 변경

    • 그룹 순서 변경: 그룹 이름 옆의 드래그 핸들(☰)을 이용해 그룹 목록의 순서를 바꿀 수 있습니다.
    • 스트리머 순서 변경: 그룹 내 스트리머 이름 옆의 드래그 핸들(⋮)을 이용해 스트리머의 순서를 변경할 수 있습니다.
  • person

    스트리머 추가 및 삭제

    • 추가: 팔로잉 목록의 스트리머를 끌어다 놓아 그룹에 쉽게 추가합니다.
    • 삭제: 그룹 내 스트리머를 마우스 우클릭하여 삭제할 수 있습니다.

그룹 관리 모달에서 가능한 기능

'내 그룹' 영역의 '관리' 버튼을 클릭하면 모달 창이 열립니다.

  • edit

    그룹 생성 및 편집

    • 새로운 그룹을 만들고 이름을 지정합니다.
    • 만들어진 그룹의 이름을 수정하거나 삭제할 수 있습니다.
  • person_add

    스트리머 관리

    • ID로 추가: 스트리머 ID를 직접 입력하여 원하는 그룹에 추가할 수 있습니다.
    • 삭제: 그룹 내 스트리머 옆의 삭제 버튼(🗑️)으로 삭제할 수 있습니다.

공통 설정 (팝업 창에서 가능)

확장 프로그램 아이콘을 클릭하여 팝업 창을 열어 설정할 수 있는 기능입니다.

  • visibility_off

    메뉴 숨기기

    원하는 메뉴를 토글하여 '추천 채널', '파트너 스트리머', '서비스 바로가기'를 숨길 수 있습니다.

  • swap_vert

    그룹 위치 변경

    그룹 목록을 '팔로잉 채널' 목록 위 또는 아래에 표시합니다.

  • folder_open

    데이터 백업 및 복원

    그룹 데이터를 JSON 파일로 내보내고 가져올 수 있습니다.
